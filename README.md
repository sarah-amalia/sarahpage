# Sarah Amalia — Technical Writer Portfolio

Personal portfolio and writing showcase built with Astro.

## Features
- CV & work experience
- Projects & writing samples
- Blog / free writing
- Dark/light mode

## Built with
- [Astro](https://astro.build)
- [DaisyUI](https://daisyui.com)
- [Tailwind CSS](https://tailwindcss.com)
- Based on [astro-cv-esquelete](https://github.com/mmouzo/astro-cv-esquelete)
